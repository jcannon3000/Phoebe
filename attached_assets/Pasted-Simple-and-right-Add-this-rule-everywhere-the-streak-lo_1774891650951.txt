**The rule**

A window counts toward the streak only if at least two circle members posted within that window.

One person showing up alone does not increment the streak. It does not break the streak either. It simply does not count.

The streak measures the circle showing up together — not individual participation.

---

**Updated streak logic**

```
After each window closes:

Count posts for that window_date on this shared_moment_id.

If posts >= 2:
  Window is a BLOOM
  Increment current_streak by 1
  Update longest_streak if current_streak > longest_streak
  Increment total_posts
  Update last_post_date
  If state was needs_water or dormant → return to previous state

If posts == 1:
  Window is a SOLO
  Streak neither increments nor resets
  State unchanged
  Post is saved and visible in journal
  Individual streak for that user increments
  Display note: "Only you showed up this time 🌿"

If posts == 0:
  Window is a WITHER
  First consecutive wither → state: needs_water, streak pauses
  Second consecutive wither → state: dormant, streak resets to 0
```

**Why this rule matters**

The streak is not a personal achievement. It is proof that people keep showing up for each other. One person posting alone every day for thirty days is admirable but it is not the same thing as two people showing up together. The streak measures the relationship. The relationship requires at least two.

This rule also creates a gentle social dynamic that nudges participation without being extractive. If you know your friend already posted, you feel the pull to show up so the streak counts. That pull is healthy — it is the same pull that makes a running partner more effective than a solo training plan. You are not accountable to an app. You are accountable to a person.
